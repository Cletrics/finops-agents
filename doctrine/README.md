# Doctrine

Short principle docs that every agent in this repo should assume you've
read. These are the cross-cutting ideas that sharpen agent behavior
across every Capability.

- [Iron Triangle](./iron-triangle.md) -- cost is never free of trade-offs against speed and quality
- [Data in the Path](./data-in-the-path.md) -- outputs that don't reach the stakeholder's existing workflow don't change behavior
- [Crawl, Walk, Run](./crawl-walk-run.md) -- maturity is per-Capability, not per-org
- [FCP Canon Anchors](./fcp-anchors.md) -- quotes and stories worth citing by name

Agents reference these by path (e.g., `../doctrine/iron-triangle.md`)
rather than repeating the content. When in doubt, an agent that is
about to recommend an action should read its own doctrine pointers
first.
